設定 OAuth Client ID

  1. 前往 Google Cloud Console
  2. 建立專案 → API與服務 → 啟用 Google Sheets API 與 Google Drive API
  3. 憑證 → 建立 OAuth 2.0 用戶端 ID → 類型選「網頁應用程式」
  4. 授權的 JavaScript 來源填入此頁面的網址（本機測試填 http://localhost）
  5. 複製 Client ID 貼入下方並儲存
